 Royal Chai – Coming Soon Landing Page 

A premium “Coming Soon” landing page designed for **Royal Chai**, an instant chai tea premix inspired by the timeless traditions of royal Indian households.

 Brand Concept
Tagline: Brew the Royal Tradition  
Royal Chai blends heritage, warmth, and indulgence into a modern digital experience, reflecting luxury and authenticity in every detail.

 Features
- Elegant royal-themed UI (gold, deep brown, cream tones)
- Responsive design (mobile & desktop friendly)
- Email notification signup section
- “Launching Soon” call-to-action
- Social media icons
- Clean, minimal, and modern layout

 Tech Stack
- HTML5
- CSS3
- Font Awesome (icons)
- Google Fonts (Playfair Display & Poppins)

Design Choices
The design uses warm brown gradients and gold accents to convey royalty, tradition, and premium quality. Serif typography enhances the heritage feel, while subtle animations and spacing ensure a modern, elegant user experience.
